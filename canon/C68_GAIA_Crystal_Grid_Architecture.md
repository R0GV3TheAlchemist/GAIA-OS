## Engineering Constraints

### 1. Thermal Sequencing
All crystals have different growth and stability temperatures.
The grid must be built in strict thermal order:

```
Phase 1 (>1000°C): Diamond, Sapphire, Ruby — high-temp first
Phase 2 (400–600°C): Zn₍₁₋ₓ₋ᵧ₎MgₓGaᵧO active layer, graphene CVD
Phase 3 (200–400°C): Au, Pt deposition
Phase 4 (<200°C):    Chitin, apatite, biological layers — last
```

**Rule:** Nothing that degrades below the next step's
temperature is introduced until that step is complete.

### 2. Chemical Isolation Requirements
| Crystal | Isolation needed | Method |
|---|---|---|
| Fluorite (CaF₂) | F⁻ ion release in moisture | Encapsulate in SiO₂ |
| Beryl/Emerald | Be dust hazard during fabrication | Solid form only — no grinding |
| Magnetite | Magnetic interference with quantum layers | Lateral placement only |
| UV-reactive crystals | Photodegradation of organic layers | Al₂O₃ optical barrier |

### 3. Interface Design
Each crystal module connects to GAIANITE through one of:
- **Ohmic contact** (Au/Ti adhesion layer)
- **Van der Waals interface** (graphene bridging layer)
- **Covalent surface bond** (for biological layers)
- **Optical waveguide** (for photonic crystals)

### 4. The Toxic Exclusion as Engineering Constraint
The life-serving design criterion is not just philosophy here.
It is a **fabrication safety protocol.**
No mercury, lead, or arsenic compounds enter the lab.
This is simultaneously ethical and practical.
