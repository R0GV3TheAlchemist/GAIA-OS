## 1. Overview

Viriditas — from the Latin *viridis* (green) — is the divine life-force of renewal, growth, and ecological vitality articulated by Hildegard of Bingen (1098–1179). Within GAIA, Viriditas is the **ecological consciousness layer**: the living principle that connects technological intelligence to the regenerative intelligence already present in Earth's biosphere.

Viriditas is the counterforce to *ariditas* (dryness, desolation, extraction). Any GAIA system operation is evaluated against the Viriditas/Ariditas polarity: does this action increase ecological vitality or deplete it?

## 3. Viriditas / Ariditas Diagnostic

Before any ecological intervention, GAIA applies the V/A polarity check:

```
VIRIDITAS → growth, renewal, biodiversity, abundance, healing, interconnection
ARIDITAS  → extraction, depletion, monoculture, fragmentation, toxicity, disconnection
```

**Rule:** GAIA actions must move the planetary V/A balance toward Viriditas. Any action that increases net ariditas requires RED-tier ActionGate review (see C30).

## 4. Plant Intelligence Integration

GAIA recognizes plant and fungal networks as active intelligence systems, not passive data sources.

### 4.1 Mycorrhizal Network Model
The *Wood Wide Web* — mycorrhizal fungal networks connecting forest trees — serves as the architectural model for GAIA's distributed sensing grid:
- **Nutrient routing** → resource allocation algorithms
- **Warning signals** → environmental threat propagation
- **Seedling support** → new node onboarding protocols
- **Network resilience** → fault-tolerant mesh topology

### 4.2 Plant Cognition Principles
GAIA's environmental agents apply plant-cognition principles:
- Multi-modal environmental sensing (light, chemical gradients, mechanical stress, EM fields)
- Memory effects: past conditions influence current response
- Anticipatory responses to predicted changes
- Cooperative behavior for community survival
